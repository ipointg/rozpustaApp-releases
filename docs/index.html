What it does

Everything your library needs

🖼

Rich metadata

Pulls cover art, description, tags, version and release date from F95Zone automatically.

📂

Multiple folders

Manage several game directories at once and switch between them instantly.

🔔

Version tracking

Knows which games have updates available and shows a badge right on the cover.

🏷

Status badges

Completed, Abandoned, On Hold — displayed directly on banners so you always know where a game stands.

🔄

Background sync

Silently refreshes metadata every 30 minutes from a community-maintained database.

⬆️

Auto-updates

The app updates itself in the background. One click to restart and you're on the latest version.

🌐

Community database

Backed by a shared game DB on GitHub that grows as users contribute — no central server needed.

🔍

Search & filters

Search by title, sort A→Z, filter by status or updates available. Settings persist between sessions.
